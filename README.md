## Titlul Proiectului
**Analiza Corelatiilor Canonice (CCA) privind Impactul Indicatorilor Educationali asupra Starii de Sanatate la Nivel Global**

### Descriere GeneralÄƒ
In cadrul lucrarii este aplicata **Analiza Corelatiilor Canonice (Canonical Correlation Analysis - CCA)**. Acest proiect a fost realizat pentru disciplina **DSAD** si exploreaza relatia multivariata dintre doua domenii fundamentale ale dezvoltarii umane: **Educatia** si **Sanatatea**.

### Obiectiv Principal
Obiectivul principal este de a identifica in ce masura un set de indicatori educationali poate explica si prezice variatia unui set de indicatori de sanatate .
